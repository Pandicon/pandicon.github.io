Information
Musicon is a music bot for discord that allows you to play music, playlists of songs, pause and resume the songs and much more and is open-source. You can find the Github repository here. You can download the code and have some fun with it or host it yourself, but I decided that I will host the most up-to-date version myself. You can invite the bot to your server here.

Musicon is a music bot for discord that allows you to play music, playlists of songs, pause and resume the songs and much more. It was made for my computer science class and is open-source. You can find the Github repository here. I will probably not host it as my own bot, but you can have some fun with it if you download the code and run it yourself or even edit it.

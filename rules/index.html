Rules
1. Using any auto-typer, script, macro or any other way to automate commands is strictly forbidden.
2. Sharing exploits or game-breaking bugs with other users is forbidden. Report these to bot staff (you can find them in the support server). In addition, if you come across a not harmful bug, you should report it in the bug reports channel in the support server (for example the bot showing wrong member count of a server).
3. Abusing any exploits or game-breaking bugs is also strickly forbidden.
4. You shouldn't require users to pay you to be able to use the bot or enter giveaways for the bot. The only exception to this are booster perks. Using level roles as a requirement is perfectly fine, even if you use a different bot for that.
5. Using the bot to spread racism, sexism, honophobia etc. is strictly forbidden.
6. You shouldn't use the economy system to advertise anything, for example paying people economy items if they join your server is forbidden.
7. Buying any bot stuff with real life currency or discord nitro is forbidden. Buying anything costing real money (regarding the bot) from unofficial sources is forbidden.
8. You shouldn't cause any unnecessary drama, start any harmful rumors about the bot or witch hunt bot staff.
9. Using the bot for anything harmful/illegal is strictly forbidden.
10. You shouldn't argue about the actions that might be taken against you regarding the bot by staff members. Note that you can always appeal, but you shouldn't prolong the process on purpose.
11. By using the bot, you accept the Discord Terms of Service and Community Guidelines.
Punishments
Failing to follow the rules above will most likely result into punishments. Some of the possible outcomes are listed below.
Verbal warning: This is the lowest punishment and it will be used if you break a rule unintentionally and/or you breaking the rule didn't cause any damage.
Bot ban: This punishment allows us to adjust it depending on our needs. If you cause minor damage, you might get bot banned for a day. The automod automatically bot bans you for 12 hours if you fail to verify that you aren't using some sort of autotyper. The usual bot ban is only for 'grindable' and easily 'macroable' commands, such as economy ones. You might also find yourself in a total bot ban, which will restrict you from using any commands. This will be usually used if you abuse an exploit or lag the bot intentionally.
Data removal: If we find out that you used a macro to grind some economy items, we might remove them. However, if you continue to use some sort of automation to get items in the bot, we might as well remove all your data from the bot (all economy items, pats, hugs etc.). We can't remove per-server data, such as levels.
